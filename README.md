## 📖 About The Project

A modern, fully responsive **personal portfolio website** built to showcase my skills, projects, and professional experience. The website features a clean UI, smooth animations, dark/light theme support, and an interactive user experience.

### ✨ Key Features

- 🎨 **Modern & Clean Design** - Professional gradient-based UI
- 📱 **Fully Responsive** - Works perfectly on all devices
- 🌓 **Dark/Light Theme** - User preference with localStorage persistence
- ⚡ **Smooth Animations** - Scroll-based and hover effects
- 📊 **Dynamic Filtering** - Portfolio projects with category filter
- 📝 **Testimonial Modal** - Interactive testimonial popup
- 📧 **Contact Form** - With validation and feedback
- 🔍 **SEO Optimized** - Semantic HTML structure
- 🚀 **Performance Optimized** - Fast loading with lazy loading
